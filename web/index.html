N-Body Simulator
Python Data Structures OOP Physics

This physcs-based Python script incorporates numeric integration methods to simulate the motion of the 3 body problem, and can scale all the way to an arbitrary number, n, bodies. The program has a fully functional GUI that allows the user to change simulation variables, such as the gravitational constant and the numeric time step, and change intial conditions to see how the system evolves over time. See more at the Github.

Custom RISC-V CPU
RTL Digital Design Verilog

This project implements a 3-stage pipelined RISC-V processor (Fetch/Decode → Execute → Memory/Writeback) supporting the complete RV32I instruction set with data forwarding networks to resolve hazards between pipeline stages. The processor integrates with separate 4KB direct-mapped write-back instruction and data caches that interface with external memory through a multi-cycle FSM-based controller handling cache hits, misses, and dirty line evictions. This design is fully synthesized with place-and-route for ASIC implementations and hardware verified via synthesis to FPGA.
