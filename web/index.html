N-Body Simulator

This physcs-based Python script incorporates numeric integration methods to simulate the motion of the 3 body problem, and can scale all the way to an arbitrary number, n, bodies. The program has a fully functional GUI that allows the user to change simulation variables such as the gravitational constant and the numeric time step to explore how these parameters factor into the simulation.
