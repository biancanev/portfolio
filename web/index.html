N-Body Simulator
Python Data Structures OOP Physics

This physcs-based Python script incorporates numeric integration methods to simulate the motion of the 3 body problem, and can scale all the way to an arbitrary number, n, bodies. The program has a fully functional GUI that allows the user to change simulation variables, such as the gravitational constant and the numeric time step, and change intial conditions to see how the system evolves over time. See more at the Github.
